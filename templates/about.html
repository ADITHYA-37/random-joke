About Random Joke

Bringing laughter with tech and family-friendly jokes!

What is this app?

The Random Joke is a fun web application built with Flask. It provides you with randomly generated jokes in two categories: tech jokes for the coding enthusiasts and family-friendly jokes for everyone to enjoy.

Features

  • Generate random tech jokes instantly
  • Get family-friendly jokes suitable for all ages
  • Dark mode friendly interface
  • Copy jokes to clipboard with one click
  • Responsive design for desktop and mobile

How it works

Select your preferred joke type (Tech Joke or Family Friendly) from the dropdown menu on the home page, click "Get Random Joke," and a random joke will appear instantly. You can then copy and share it with friends and colleagues to spread the laughter!
